### History

When this developer's College Alumni group IIMA79 (Indian Institute of 
 Management, Ahmedabad, Batch of 1979) decided to migrate to Slack, they
 chose to restrict total storage to 5GB (so as to avail of Slack's 'free
 tier') by periodically moving old content to an external website.
 
 This code implements the necessary Slack API functionality for the
 actual transfer program.
 
 (About that name: Obviously, "Cull De Slack" means a cull of aged-out
 Slack content. And if you hail from Punjab, India, it  sounds downright 
 futuristic; "Kal Di Slack" = "Tomorrow's Slack". But less obviously it
 is an acronym for "CB Uncle's Lovely Little Doorway to Enter Slack"!).
